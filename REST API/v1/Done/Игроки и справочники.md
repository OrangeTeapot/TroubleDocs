### GET /api/v1/players
Поиск игроков (для приглашений в идею). **Auth:** сессия. **Запрос:** `?q=` (≥2 симв.).
- **Ответ (`result`):** `{ id; displayName; username; avatarUrl; hue:number }[]` — только
  публичные профили (`publicProfileWhere`), без себя, до 20.

### GET /api/v1/languages
Публичный справочник языков. **Auth:** нет. **Ответ (`result`):** `{ code; name }[]`
(ISO 639-1 + локализованные имена через `Intl.DisplayNames`).
