### GET /api/v1/profile/games
- **Auth:** сессия. **Ответ (`result`) — `ProfileGameDTO[]`** (cap 500, сорт по createdAt desc):
  ```ts
  { id:string; gameId:string; name:string; iconUrl:string|null;
    steamAppId:number|null; developer:string|null; publisher:string|null; isFree:boolean;
    preference:"WONT_PLAY"|"DONT_WANT"|"CAN_PLAY"|"WANT_TO_PLAY"; playtimeMinutes:number;
    sources:{ source:GameSource; hasLicense:boolean }[];
    platforms:string[]; stores:string[]; minPlayers:number|null; maxPlayers:number|null }
  ```
  где `GameSource = STEAM|EPIC|GOG|EA|UBISOFT|BATTLENET|XBOX|PLAYSTATION|NINTENDO|ITCH|OTHER`.

### POST /api/v1/profile/games
- **Auth:** сессия. **Запрос (JSON, `addGameSchema`):**
  идентификация — `steamAppId:number` ИЛИ `name:string`; метаданные (опц.)
  `iconUrl` (safe-URL), `developer`, `publisher`, `isFree`, `igdbId`, `platforms` (≤16),
  `stores` (≤16), `minPlayers`, `maxPlayers`; владение (обяз.)
  `sources: { source:GameSource; hasLicense:boolean }[]` (≥1).
- **Ответ (`result`):** `ProfileGameDTO` (201). **Заметки:** advisory-lock по nameNormalized
  против дублей `Game`; авто-дозаполнение из Steam appdetails/IGDB.

### PATCH /api/v1/profile/games/[id]
Своя игра. **Запрос (JSON, `updateGameSchema`, union):**
`{ preference }` | `{ action:"addSource", source, hasLicense }` |
`{ action:"removeSource", source }` | `{ source, hasLicense }` (legacy-тоггл лицензии).
**Ответ:** обновлённый `ProfileGameDTO`.

### DELETE /api/v1/profile/games/[id]
Удалить игру из библиотеки (каталог `Game` не трогается). **Ответ:** `{ id: string }`.
