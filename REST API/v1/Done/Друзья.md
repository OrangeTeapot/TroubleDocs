### GET /api/v1/friends
- **Ответ (`result`):** `{ friends: FriendDTO[]; incoming: FriendDTO[]; outgoing: FriendDTO[] }`,
  `FriendDTO { friendshipId; profileId; slug; displayName; username; avatarUrl; createdAt }`.
  `friends`=ACCEPTED; `incoming`=PENDING ко мне; `outgoing`=PENDING от меня.

### POST /api/v1/friends
- **Запрос (JSON, `friendActionSchema`):** `{ profileId; action: "request"|"accept"|"decline"|"remove" }`.
- **Ответ (`result`):** `{ state: "none"|"friends"|"outgoing"|"incoming" }`.
- **Ошибки:** 409 действие невалидно для текущего состояния; 422 self/таргет не найден.
- **Заметки:** `request` авто-принимает встречную заявку; гонка пары разрешается через
  unique `pairKey`; `remove` снимает дружбу/заявку в любом направлении (рвёт доступ к DM).
