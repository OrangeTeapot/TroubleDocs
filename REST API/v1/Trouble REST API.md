Справочник по всем HTTP-эндпоинтам приложения. Актуально для ветки **beta (v0.68.3)**.
Документирует кастомные роуты `app/api/**`. Внешние редиректы (Steam/Xbox OAuth) и вебхук
CloudPayments описаны отдельно, т.к. не следуют общему конверту.

> Источник правды — код в `app/api/**` + Zod-схемы в `lib/validation/**` и сериализаторы
> `lib/**/serialize.ts`. При расхождении доверять коду.
