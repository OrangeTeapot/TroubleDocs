## 3. Описание Сервиса
Платформа представляет собой веб-сервис для игрового сообщества, обеспечивающий, в частности:

- поиск напарников для совместной игры;
- создание и публикацию профилей игроков;
- создание игровых команд и публикацию их профилей;
- организацию совместных игровых сессий через создаваемые пользователями объявления о наборе («идеи» встреч);
- работу рекомендательных систем подбора игроков и игр;
- общение между пользователями;
- предоставление платных услуг (см. [[#4. Платные услуги и приём платежей|раздел 4]]).

Сервис может функционировать для пользователей по всему миру. Доступ к Сервису может осуществляться по приглашениям (в период ограниченного тестирования) либо в открытом режиме.

## 4. Платные услуги и приём платежей
На Платформе могут предоставляться следующие платные услуги:

- **Премиум-подписка «More Trouble»** — расширенный набор возможностей Сервиса, оформляемый на период (месяц или год) с возможностью автоматического продления (рекуррентные платежи);
- **Продвижение профиля** (Profile Boost) — повышение видимости профиля пользователя;
- **Продвижение команды** (Team Promotion) — повышение видимости профиля команды;
- **Размещение рекламы** (Advertising) — размещение рекламных материалов на условиях Оператора.

Приём платежей осуществляется через платёжного провайдера **CloudPayments** и иных привлечённых Оператором поставщиков платёжных услуг. Оператор не получает и не хранит полные реквизиты платёжных карт пользователей; обработка платёжных данных производится на стороне платёжного провайдера в соответствии со стандартом безопасности PCI DSS.

Способы оплаты: банковская карта, Система быстрых платежей (СБП) и иные способы, поддерживаемые платёжным провайдером. Условия оказания платных услуг определяются Публичной офертой, порядок возврата — Политикой возвратов.
