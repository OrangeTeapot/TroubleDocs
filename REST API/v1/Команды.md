DTO: `TeamDTO`
```ts
{ id; slug; name; description; avatarUrl; createdAt; memberCount;
viewerRole:"OWNER"|"MEMBER"|null; members: TeamMemberDTO[] }`,
`TeamMemberDTO { profileId; profileSlug; displayName; username; avatarUrl; role }
```

### GET /api/v1/teams
Мои команды (где я участник). **Ответ:** `TeamDTO[]` (сорт createdAt desc).

### POST /api/v1/teams
- **Запрос (JSON, `createTeamSchema`):** `name` (2–60), `description` (≤500, nullable).
- **Ответ:** `TeamDTO` (создатель → OWNER). Слаг из имени (транслит + уникализация, ретрай при гонке).

### GET /api/v1/teams/[id]
Одна команда (slug или id). **Ответ:** `TeamDTO`. **Ошибки:** 404 если не участник (не палим существование).

### PATCH /api/v1/teams/[id]
Только владелец. **Запрос (`updateTeamSchema`):** `name?`, `description?`. **Ответ:** `TeamDTO`.
Слаг неизменяем. **Ошибки:** 403 не владелец.

### DELETE /api/v1/teams/[id]
Только владелец. **Ответ:** `{ deleted: true }` (участники + чат каскадно).

### POST /api/v1/teams/[id]/members
Только владелец. **Запрос (`addMemberSchema`):** `{ profileId }`. **Ответ:** `TeamDTO`
(идемпотентно по unique teamId+profileId, роль MEMBER). **Ошибки:** 403; 422 `profile_not_found`.

### DELETE /api/v1/teams/[id]/members?profileId=…
Убрать участника (по умолчанию — себя). **Ответ:** `{ left: true }` (вышел сам) или `TeamDTO`
(владелец убрал другого). **Ошибки:** 403 не-владелец убирает другого; 409 владелец не может выйти.

### POST /api/v1/teams/[id]/avatar · DELETE …/avatar
Аватар команды, только владелец. POST `multipart` → `{ avatarUrl }` (201); DELETE → `{ avatarUrl: null }`.
