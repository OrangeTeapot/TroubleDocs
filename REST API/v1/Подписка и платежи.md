### GET /api/v1/subscription
- **Ответ (`result`):** `{ plan:"more_trouble"; active:boolean; currentPeriodEnd:string|null }`
  (`active` = status active И (end null ИЛИ в будущем); null end = бессрочно).

### POST /api/v1/subscription/cloudpayments/pay
Вебхук CloudPayments (**публичный, защита HMAC; НЕ использует конверт `{result,meta}`**).
- **Запрос:** `application/x-www-form-urlencoded` — `AccountId`, `TransactionId`, `Amount`,
  `Currency` (RUB), `Data` (JSON `{userId?}`); заголовок `Content-HMAC` = Base64(HMAC-SHA256(rawBody, secret)).
- **Ответ:** `{ code: 0 }` (принято/активировано) | `{ code: 13 }` (отбой → CloudPayments повторит).
- **Заметки:** HMAC по сырому телу; идемпотентность по `ProcessedPayment(provider,transactionId)`;
  период по сумме (month=300 / year=3000 RUB ±0.5), продление от текущего конца.

### POST /api/v1/subscription/cloudpayments/dev-activate
Только для локального теста, при `CLOUDPAYMENTS_DEV_ACTIVATE=true` (иначе `404`).
- **Auth:** сессия. **Запрос (JSON):** `{ period?: "month"|"year" }` (деф. month).
- **Ответ (`result`):** `{ active:true; currentPeriodEnd:string|null }` (без HMAC/вебхука; provider `cloudpayments-dev`).
